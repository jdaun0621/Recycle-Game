# Recycle Game / 수박 클론 게임

이 프로젝트는 [moonfloof](https://github.com/moonfloof)님의 수박게임을 기반으로 수정하였습니다.


---

## 사용 기술

- HTML, 
- CSS 
- JavaScript (ES6+)
- [matter.js](https://github.com/liabru/matter-js) — 2D 물리 엔진 라이브러리
